# Smart Parking

Frontend written in React + Typescript.

Backend writtein in NodeJS (ExpressJS).

### Run frontend

```bash
cd frontend
npm i
npm run dev
```

### Run backend

IMPORTANT: You need to setup .env file in this directory like this:

```bash
MQTT_USERNAME=x
MQTT_PASSWORD=x
```

```bash
cd backend
npm start
```

### Testing publishing

IMPORTANT: You need to setup .env file in this directory like this:

```bash
MQTT_USERNAME=x
MQTT_PASSWORD=x
```

```bash
cd publish_test
npm start
```
